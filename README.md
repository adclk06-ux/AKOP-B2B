# AKOP B2B Enterprise

Final AKOP B2B enterprise React/Vite project.

## Local Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel Settings

- Framework: Vite
- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`

Do not commit `node_modules`, `.env`, `.vercel`, or `dist`.
